0. Es OpenSource -- Lo ha implementado Terracota -- GestionEhCache.
1. Se levanta un puerto RMI dispuesto para recibir peticiones. No hay que instalar ningun servidor de aplicaciones (Tomcat-Weblogic..)
2. Se accede a cada Job de Talend de una manera natural (importando el .jar y accediendo a las clases y metodos).
3. Potente API de gestion de tiempos y recursos (cuando y periodicidad de lanzamientos, maxima capacidad del pool de ejecuciones etc)
4. Utiliza por Persistencia OPENJPA

Resumen. Esta hecho en Java y la flexibilidad e integracion es casi total (se pueden añadir nuevos plugins -addons etc)
Además se puede programar como un servicio. No es necesario la intervencion humana cada vez que se reinicia el equipo.
